OrderBud

OrderBud is a mobile app marketplace that helps users find small businesses in their neighbourhood that sell products they are looking for. Users can explore the menus of businesses and purchase products from them through the app.

React Native · Redux · Node.js · Express.js · MongoDB

Flasback

Flashback is a photo repository that enables you to upload individual or collections of memories and tag them with details you associate with that memory. This tagging powered a search-rank algorithm that allowed users to search memories as they had remembered them using natural language.

React.js · Node.js · Express.js · PostgreSQL

Smart Brain

This is a web application I built as the final project of the The Complete Web Developer Zero To Mastery course on Udemy. It is an app that can detect faces in images! Simply provide the URL to an image online, and the program will find the faces in the picture.

React.js · Node.js · Express.js · PostgreSQL · Clarfai Face Detection API
