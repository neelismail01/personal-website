Smart Brain This is a web application I made that can detect faces within images! Simply provide the URL to an image online, and the program will find the faces in the picture. Technologies used include: React.js, Node.js, Express.js, SQL, PostgreSQL, and the Clarifai API
YourSpace This is a Twitter-like social media site I made as a fun side project. You can create your own account, post messages and pictures, search and view posts made by other people, and like posts. Technologies used include: HTML/CSS, Materialize.css, PHP, and SQL
The 1953 Iranian Coup An analysis of British and American motives for orchestrating the military coup that overthrew Iran's first democratically elected President.
The Molotov-Ribbentrop Pact An analysis of the motives that led the Soviet Union to sign a non-aggression pact with Germany at the onset of World War 2.
The Patriot Act An analysis of whether the Patriot Act was a lawful measure to enhance counterterrorism efforts after the tragic 9/11 attacks.
